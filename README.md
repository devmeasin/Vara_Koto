## Run Locally

Clone the project

```bash
  git clone https://github.com/easinwebpro/
```

Go to the project directory

```bash
  cd Vara_Koto
```

Install dependencies

```bash
  yarn or npm i
```

Start the server

```bash
  ionic serve
```

**Open [http:///localhost:8100](http://localhost:8100) to view it in your browser.**
